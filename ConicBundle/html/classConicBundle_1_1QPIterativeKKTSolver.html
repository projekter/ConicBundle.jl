Detailed Description

Iterative solver for the full symmetric, in general indefinite primal dual KKT System within QPSolverBasicStructures.

See the text to QPKKTSolverObject for the terminology of the primal dual KKT System and the general outline.

This implements the matrix-vector mutliplication of CH_Matrix_Classes::IterativeSystemObject for the entire system consisting of all blocks H, A, B, C.

It also serves as a kind of base class for specialized variants that may form partial Schur complements in representing the system, an initial example is the class QPIteratvieKKTHASolver

The most important routines of the model described in the QPModelBlockObject that are required here are (besides sizes and multiplications with the bundle matrix B)

  • QPModelBlockObject::add_localrhs() for constructing the right hand side
  • QPModelBlockObject::computed_step() for communicating the solution step to the model
  • QPModelBlockObject::localsys_mult() for the multiplication with the B and C blocks on the main diagonal
