Detailed Description

implements a Gram matrix as Coeffmat for a sparse rectangular CH_Matrix_Classes::Sparsemat (for use with MatrixSDPfunction, see implemention of a PSCOracle (PSCAffineFunction)).

While a Gram matrix by itself is always positive semidefinite, there is an extra flag for using its negative version.
