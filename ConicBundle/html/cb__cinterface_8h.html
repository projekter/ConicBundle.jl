Detailed Description

Header declaring Interface to ConicBundle for language C.

Version
1.0
Date
2005-03-01
Author
Christoph Helmberg

indices start at 0!

solves min_{y in R^m} f_0(y) + f_1(y) + ... + f_k(y)

for convex functions f_i, the y-variables may be bounded or box constrained
