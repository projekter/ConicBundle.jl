Detailed Description

Implementation of MinorantExtender for NNCBoxSupportFunction.

This object will be returned as an object on the heap by NNCBoxSupportFunction::apply_modification and will be deleted by ConicBundle after use. Its purpose is to fill up further coordinates of the minorant (if this is possible).
