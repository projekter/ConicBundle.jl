Detailed Description

implements a Gram matrix with zero diagonal as Coeffmat for a sparse rectangular CH_Matrix_Classes::Sparsemat (for use with MatrixSDPfunction, see implemention of a PSCOracle (PSCAffineFunction)).

As for CMgramsparse there is an extra flag for using the positive or the negative version of the matrix.
